## Game Description

**Dhaka Retro Metro** is a high-octane, Naruto-themed action-platformer built using the **iGraphics** library in C++. The project demonstrates advanced graphics programming concepts including multi-layered scrolling, state-machine boss logic, and logic-gate puzzle integration. 

In this retro-inspired world, players take control of **Kakashi Hatake** on a mission to secure the metro from legendary Akatsuki threats. Battle your way through iconic villains, manage your ninja resources, and prove your intellect to save the city.

## Features
- **Epic Boss Rush**: Battle legendary foes including Orochimaru, Deidara, Itachi, and Sasori across 4 unique levels.
- **Advanced Combat**: Master frame-perfect animations for knife strikes, kunai throws, and high-damage Special Jutsus.
- **Intellectual Gates**: Prove your logic in a custom-built Sudoku minigame mandatory for progressing to the final arena.
- **Modern Retro Aesthetic**: A polished UI featuring neon glow effects, motion-blur loading screens, and cinematic story transitions.
- **Competitive Leaderboard**: Compete for the fastest "Ninja Time" and save your scores to a persistent leaderboard.

## Project Details
IDE: Visual Studio 2012 / 2013

Language: C, C++.

Platform: Windows PC.

Genre: 2D Action Adventure / Boss Rush.

## How to Run the Project

Make sure you have the following installed:
- **Visual Studio 2013** (or 2012 with compatible toolsets)
- **iGraphics Library** (included in this repository)
- **OpenGL/GLUT Dependencies** (included in the source folder)

### Open the project in Visual Studio
- Open Visual Studio.
- Go to **File → Open → Project/Solution**.
- Locate and select the `Dhaka_Retro_Metro.sln` file.
- Click **Build → Build Solution**.
- Run the program by clicking **Debug → Start Without Debugging** (or press `F5`).

## How to Play

### **Controls**
| Action | Primary Key | Alternate |
| :---: | :---: | :---: |
| **Move Left** | `A` | `Left Arrow` |
| **Move Right** | `D` | `Right Arrow` |
| **Jump / Double Jump** | `W` | `Space` / `Up` |
| **Crouch / Drop** | `S` | `Down Arrow` |
| **Knife Strike** | `K` | - |
| **Throw Kunai** | `J` | - |
| **Special Jutsu** | `O` | - |
| **Run Mode** | `L` (Hold) | - |
| **Story Pager** | `P` | - |
| **Menu / Back** | `ESC` | `M` |

### **Game Rules**

- **Health & Lives**: Players start with 100 HP and 3 Lives.
- **Damage Mechanics**:
    - **Knife Strike**: High-impact melee attack (34 Damage).
    - **Kunai Throw**: Ranged tactical attack (25 Damage / 10 to Bosses).
- **Scoring**: Your rank is determined by your "Ninja Time" (completion speed).
- **Sudoku Gate**: Accessing the final level requires solving a tactical Sudoku puzzle.
- **Boss Phases**: Villains enter "Rage Modes" and change attack patterns as their health drops.
