# Dhaka Retro Metro 🎮

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform: Windows](https://img.shields.io/badge/Platform-Windows-blue.svg)](https://www.microsoft.com/windows)
[![Language: C++](https://img.shields.io/badge/Language-C%2B%2B-00599C.svg)](https://isocpp.org/)

**Dhaka Retro Metro** is a high-octane, Naruto-themed action-platformer built using **C++** and the **iGraphics** library. Take control of **Kakashi Hatake** as you battle through iconic villains, solve strategic puzzles, and navigate a neon-retro aesthetic world inspired by both the cyberpunk vibe and classic ninja lore.

---

## ✨ Features

- **Dynamic Combat System**: Master Kakashi's arsenal including Knife Strikes, Kunai Throws, and high-damage Special Jutsus.
- **Epic Boss Battles**: Challenge legendary foes like **Deidara**, **Sasori**, and **Itachi** across 4 unique levels.
- **Logic Gates**: Progress through Level 3 by proving your intellect in a custom-built **Sudoku Minigame**.
- **Modern Retro Aesthetic**: Experience a polished UI with neon glow, glassmorphism, and smooth transitions.
- **Competitive Leaderboard**: Track your fastest "Ninja Time" and compete for the top rank.
- **Full Audio Suite**: Immersive BGM and SFX with toggleable settings.

## 🎮 How to Play

| Action | Keys |

## 🧠 Sudoku Minigame

To pass the gate in Level 3, you must solve a Sudoku puzzle:
- **Navigate**: Use Arrow Keys.
- **Input**: Press `1` - `9`.
- **Clear**: Press `0` or `Backspace`.
- **Verify**: Press `Enter` once the grid is complete.

## 🚀 Getting Started

### Prerequisites
- **Operating System**: Windows
- **IDE**: Visual Studio (2012 or later recommended)
- **Dependencies**: iGraphics (included), OpenGL (GLUT), Glaux.

### Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/YourUsername/Dhaka-Retro-Metro.git
    ```
2.  **Open Project**:
    Navigate to `DhakaRetroMetro_v1.0.2` and open `Dhaka_Retro_Metro.sln` in Visual Studio.
3.  **Build Settings**:
    - Ensure the configuration is set to `Debug` or `Release` (x86).
    - Ensure `glut32.dll` is in the output directory (already present in the source folder).
4.  **Run**:
    Press `F5` or the **Start** button in Visual Studio.
